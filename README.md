# sgi-fabricajas
SGI para Fabricajas del Eje S.A.S.. Plataforma web en PHP, JS y MySQL que digitaliza el control de inventario de cartón. Incluye gestión de múltiples bodegas, reportes estratégicos y niveles de stock en tiempo real. Posee roles de Admin, Logística y Bodega para garantizar trazabilidad y seguridad .
