# Descripción General
Este proyecto consiste en el desarrollo de una plataforma web integral diseñada para modernizar y optimizar el control de inventarios en Fabricajas del Eje S.A.S.. El sistema surge como solución a la dependencia de registros manuales y hojas de cálculo, proporcionando una herramienta centralizada que garantiza la precisión de los datos y la visibilidad del stock en tiempo real. 

El SGI permite la gestión digitalizada de materiales (cajas y láminas de cartón), eliminando errores humanos y mejorando la eficiencia operativa en los procesos de entrada y salida de mercancía. 
Stack TecnológicoBackend: PHP.  Base de Datos: MySQL.  Frontend: HTML, CSS (Tailwind CSS) y JavaScript. 

Gestión de Roles y PermisosEl sistema implementa un control de acceso basado en roles para garantizar la seguridad y la integridad de la información:

Administrador: Control total del sistema y gestión de usuarios.  

Encargado de Logística: Supervisión del inventario, acceso a reportes detallados y ajustes de discrepancias.

Usuario de Bodega: Registro físico de las entradas y salidas de materiales.

Funcionalidades PrincipalesControl de Movimientos: Registro automatizado de entradas y salidas con trazabilidad completa (fecha, hora, cantidad y usuario responsable).

Gestión Multibodega: Capacidad para crear y administrar diferentes bodegas, asociando productos a ubicaciones específicas.

Validaciones en Tiempo Real: Bloqueo automático de salidas si no hay stock suficiente y alertas de niveles mínimos de inventario.

Reportes Estratégicos: Generación de informes de estado actual e histórico de movimientos para la toma de decisiones informadas.

Auditoría: Registro de la actividad de cada usuario para un control total sobre las modificaciones en el sistema.
